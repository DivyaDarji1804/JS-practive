this whole section is about loop concepts :
learn if-else, loop,forin,forof,array methods 

⭐ 1. forEach()
Purpose:

Run a function on each element (perform some action).

Important:

❌ Does NOT return a new array

❌ Does NOT modify the original array unless you do it manually

✔ Used only for side effects (logging, updating UI, etc.)

numbers.forEach(n => console.log(n));

⭐ 2. filter()
Purpose:

Select items that match a condition.

Important:

✔ Returns a new array

✔ Only keeps items where condition === true

❌ Does NOT change original array

const evens = numbers.filter(n => n % 2 === 0);

⭐ 3. map()
Purpose:

Transform each element into a new value.

Important:

✔ Returns a new array

✔ Same length as the original

❌ Does NOT filter

❌ Does NOT modify original array

const doubled = numbers.map(n => n * 2);


⭐ 4. reduce()
Purpose:

Reduce the array to one single value.

Important:

Uses accumulator (acc)

Can create sum, product, object, array, or anything

Very powerful and flexible

const sum = numbers.reduce((acc, n) => acc + n, 0);



| Method      | Returns        | Purpose                        | Changes Original? |
| ----------- | -------------- | ------------------------------ | ----------------- |
| **forEach** | ❌ undefined    | Run actions                   | ❌ No              |
| **filter**  | ✔ New array    | Keep items that pass condition | ❌ No              |
| **map**     | ✔ New array    | Transform each item            | ❌ No              |
| **reduce**  | ✔ Single value | Combine all items              | ❌ No              |
